Have you ever tried to create components without using any kind of framework?
If your answer sound like ``not`` you should definitely look at Web Component technology.
Web Component is a great tool to build a reusable component without using a web-framework

## Installation
1. Clone project
```
git clone https://github.com/Almazatun/noFR-web-components.git
```
2. Download dependencies
```
yarn install
```
3. Build project
```
yarn build 
```
4. Start server
```
yarn start
```
